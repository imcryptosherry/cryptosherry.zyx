GPT-4 represents a significant improvement over GPT-3.5 in terms of processing power, versatility, and capability. While GPT-3.5 is already an impressive language model, GPT-4 has the potential to be a game-changer in the field of natural language processing.

**Key Differences Between GPT-4 and GPT-3.5**

- GPT-4 has 8 times more processing power than ChatGPT (currently available to the public)
- GPT-4 is a multimodal language model that can handle text, images, audio, and video inputs
- GPT-4 can build websites from an image, a capability that GPT-3.5 does not have

[Microsoft](https://blogs.bing.com/search/march_2023/Confirmed-the-new-Bing-runs-on-OpenAI%E2%80%99s-GPT-4) has officially confirmed that the advanced GPT-4 model powers the new Bing, which features ChatGPT. While it was previously known that the updated [Bing](https://www.tomsguide.com/how-to/how-to-use-the-new-bing-with-chatgpt-and-what-you-can-do-with-it) uses a newer version of GPT-3.5, this is the first confirmation that GPT-4 has been the driving force behind the search engine. OpenAI has announced that GPT-4 will be accessible to ChatGPT Plus users and developers via the ChatGPT API. However, it is uncertain whether the free version of ChatGPT will eventually be upgraded to GPT-4.

GPT-4's multimodal capabilities allow for the use of various input types, such as YouTube clips or audio recordings, to provide answers to questions. Microsoft gave an example of how this could help businesses by automatically summarizing support calls with text after listening to recordings. This would save 500 work hours a day for a large Microsoft customer in the Netherlands that receives 30,000 calls a day that need to be summarized. However, Microsoft warns that ChatGPT will not always be reliable, even after the GPT-4 upgrade. They are working on confidence metrics to improve the chatbot's reliability.
